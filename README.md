## 🌟 Overview
**Quiz for Coders** is a high-performance learning platform designed for aspiring **MERN Stack** developers. Built entirely with **TypeScript**, this platform leverages advanced **AI** to generate real-time technical interview questions based on 2026 industry standards, ensuring type-safe and reliable practice sessions.

---

## 🚀 Key Features

* **🛡️ Type-Safe Architecture:** Developed with **TypeScript** to ensure robust code, better debugging, and scalability.
* **🔒 Secure Authentication:** Integrated **Firebase Auth** supporting Google Login and Email/Password flows.
* **🔑 Forgot Password:** A fully functional password recovery system for seamless user access.
* **🤖 AI-Generated Quizzes:** Real-time generation of Junior to Mid-level developer questions via a powerful AI engine.
* **🌐 Dual-Language Engine:** Toggle instantly between **English** and **Bengali** for both the UI and quiz content.
* **🌓 Theme Mastery:** Professional **White Mode** and a sleek **Dark Mode** toggle.
* **💾 Database Persistence:** User metadata and progress are automatically synced to the database upon login.
* **📱 Modern UI/UX:** A clean, professional layout built with **Tailwind CSS**, featuring card-based logic and micro-animations.

## 🛠️ Tech Stack

* **Language:** TypeScript (TSX/TS)
* **Frontend:** React.js, Tailwind CSS, Framer Motion

---

## 📂 How It Works (Technical Overview)

1.  **Strict Typing:** All data interfaces (User, Quiz, API Responses) are strictly typed using TypeScript to prevent runtime errors.
2.  **The Gatekeeper:** Users are redirected to a secure login page. Access to the dashboard is protected via specialized Auth Hooks.
3.  **AI Prompting:** The app communicates with the AI engine to fetch unique MERN stack questions in a strictly validated JSON format.
4.  **Real-time Feedback:** If a user misses a question, the AI provides a detailed, educational explanation of the concept instantly.
